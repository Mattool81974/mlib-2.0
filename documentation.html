MLib

Bienvenue dans MLib ! Mlib est une librairie graphique crée par un élève de terminal NSI dans un lycée du sud de la France (je ne donnerai pas plus d'information question d'anonymat). Elle permet de crée facilement et rapidement des interfaces graphiques qui raviront tous les professeurs de NSI voyont la beauté de vos projets. Une version antérieur existée déjà l'année dernière, remplacée par cette version repensée et améliorée (d'où le "MLib 2.0"). Retrouver le projet sur github : mlib-2.0.

PS : La page HTML ne contient pas encore de style, c'est pour ça qu'elle ne ressemble à rien.

Voici quelques classes présentes dans MLib :
-MWidget : la classe primaire de tout MLib.
-MApp : la classe principale dans un projet MLib.
-MFrame : une classe primaire permettant de créer des bordures.
-MText : une classe primaire permettant d'afficher et de modifier du texte.
-MImage : une classe secondaire permettant d'afficher simplement des images.
-MButton : une classe secondaire permettant de créer simplement des boutons.
-MSlider : une classe secondaire permettant de créer simplement des sliders.
-MButton : une classe secondaire permettant de créer des espaces de navigations dans un grand widget.
