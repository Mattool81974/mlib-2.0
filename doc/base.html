Base

MLib est une petite librairie pour faire des interfaces graphiques utilisant pygame. Elle fonctionne avec un système de widget. Un widget est un élément graphique de la fenêtre. L'élément fenêtre est un widget un peu différent qui contient tout les autres widgets. Il y a un système de hiérarchie où les widgets peuvent avoir des parents. Le widget enfant s'affiche dans le widget parent et adopte son système de coordonnée. Les widgets ont une taille (w, h) et une position (x, y). On peut modifier leur apparence pixel par pixel. La fenêtre principale s'occupe d'actualiser l'interface graphique.

Évènements

Les évènements sont des actions qui sont effectuées soit par l'utilisateur, soit par un widget, soit par l'OS.
Voici une liste d'évènements :
-pygame.QUIT : permet de quitter la fenêtre graphique.
