MCheckBox

MCheckBox est une classe secondaire implicite pour définir une liste de bouton cochable.
Elle hérite de MObject.
Ajouté à la version 2.3.0.

Fonctionnement

MCheckbox est une classe qui ne comporte pas d'affichage graphique. Elle s'occupe uniquement de lier des boutons pour leur permettre de fonctionner comme des boutons cochables. Il est déconseillé de modifier les boutons après leur passage dans addButton.

Fonctions

__init__()

Paramètres : app (assignée à la variable _mapp), objectType (assignée à la variable _type, basiquememnt mit à "MCheckBox")
Construit un objet MCheckBox.
Ajouté à la version 2.3.0.

addButton()

Paramètres : name (nom du bouton dans le MCheckBox), button (MButton qui correspond à ce nom)
Rajoute le bouton "button" sous le nom "name" dans les choix sélectionnables.
Ajouté à la version 2.3.0.

getActualChoice()

Paramètres : aucun
Retourne le choix actuel dans le MCheckBox.
Ajouté à la version 2.3.0.

getChangeFrameColorOnChoice()

Paramètres : aucun
Retourne le booléen de si le bouton choisie change sa couleur de bordure.
Ajouté à la version 2.3.0.

getChangeFrameWidthOnChoice()

Paramètres : aucun
Retourne la valeur du booléen de si le bouton choisie change sa largeur de bordure.
Ajouté à la version 2.3.0.

getFrameColorOnChoice()

Paramètres : aucun
Retourne la couleur de la bordure du bouton choisie si changeFrameColorOnChoice est True.
Ajouté à la version 2.3.0.

getFrameWidthOnChoice()

Paramètres : aucun
Retourne la largeur de la bordure du bouton choisie si frameWidthOnChoice est True.
Ajouté à la version 2.3.0.

isChoiceGettingChanged()

Paramètres : aucun
Retourne un tuple qui contient dans tous les cas à l'index 0 un booléen qui indique si le choix a changé à cette frame ou non.
Si le choix a changé, le tuple contient (booléen, nouveau choix, ancien choix).
Ajouté à la version 2.3.1.

setActualChoice()

Paramètres : actualChoice (nouveau choix actuel)
Change le choix actuel dans le MCheckBox par "actualChoice".
Ajouté à la version 2.3.0.

setChangeFrameColorOnChoice()

Paramètres : changeFrameColorOnChoice (nouvelle valeur du booléen de si le bouton choisie change sa couleur de bordure)
Change la valeur du booléen de si le bouton choisie change sa couleur de bordure par "changeFrameColorOnChoice".
Ajouté à la version 2.3.0.

setChangeFrameWidthOnChoice()

Paramètres : changeFrameWidthOnChoice (nouvelle valeur du booléen de si le bouton choisie change sa largeur de bordure)
Change la valeur du booléen de si le bouton choisie change sa largeur de bordure par "changeFrameWidthOnChoice".
Ajouté à la version 2.3.0.

setFrameColorOnChoice()

Paramètres : frameColorOnChoice (nouvelle couleur de la bordure du bouton choisie si changeFrameColorOnChoice est True)
Change la couleur de la bordure du bouton choisie si changeFrameColorOnChoice est True par "frameColorOnChoice".
Ajouté à la version 2.3.0.

setFrameWidthOnChoice()

Paramètres : frameWidthOnChoice (nouvelle largeur de la bordure du bouton choisie si frameWidthOnChoice est True)
Change la largeur de la bordure du bouton choisie si frameWidthOnChoice est True par "frameWidthOnChoice".
Ajouté à la version 2.3.0.

softResetObject()

Paramètres : aucun
Reset l'objet sans modifications graphiques.
Dans MCheckBox, elle permet de reset "_isGettingChanged".
Ajouté à la version 2.3.1.

_lateUpdate()

Paramètres : deltaTime (temps passé entre cette frame et la dernière frame)
Fonction appelée à chaque frame après le contrôle des éléments.
Dans MCheckBox, elle sert à détecter les clicks des boutons et adapter l'état du MCheckBox en conséquence.
Ajouté à la version 2.0.0.

Variables

actualChoice

Choix actuel selon le bouton pressé, "" si aucun bouton n'a été pressé.
Ajouté à la version 2.3.0.

baseDataByButton

Dictionnaire ayant comme clé un bouton et comme valeur les donénes de base sur le bouton.
Ajouté à la version 2.3.0.

buttonsByName

Dictionnaire ayant comme clé le nom du bouton et comme valeur le bouton.
Ajouté à la version 2.3.0.

changeFrameColorOnChoice

Booléen qui indique si le bouton choisie change sa couleur de bordure.
Ajouté à la version 2.3.0.

changeFrameWidthOnChoice

Booléen qui indique si le bouton choisie change sa largeur de bordure.
Ajouté à la version 2.3.0.

frameColorOnChoice

Couleur de la bordure du bouton choisie si changeFrameColorOnChoice est True.
Ajouté à la version 2.3.0.

frameWidthOnChoice

Largeur de la bordure du bouton choisie si changeFrameWidthOnChoice est True.
Ajouté à la version 2.3.0.

isGettindChanged

Booléen qui indique si le choix est changé pendant cette frame.
Ajouté à la version 2.3.1.

nameByButton

Dictionnaire ayant comme le bouton et comme valeur le nom du bouton.
Ajouté à la version 2.3.0.

_lastChoice

Choix qui précède le choix actuel.
Ajouté à la version 2.3.0.
