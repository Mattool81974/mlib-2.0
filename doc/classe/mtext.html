MText

MText est une classe permettant de d'afficher du texte.
Elle hérite de MFrame.

Fonctions

__init__()

Paramètres : text (assignée à la variable text), x (assignée à la variabe x), y (assignée à la variabe y), width (assignée à la variable width), height (assignée à la variable height), parent (assignée à la variable parent), widgetType (assignée à la variable _type, basiquememnt mit à "MText")
Construit un objet MText.

getCursorPosition()

Paramètres : aucun
Retourne la position du curseur dans le texte.

getCursorVisible()

Paramètres : aucun
Retourne si le curseur est visible.

getCursorWidth()

Paramètres : aucun
Retourne la largeur du curseur en pixel.

getDynamicTextCut()

Paramètres : aucun
Retourne dynamicTextCut.

getDynamicTextCutType()

Paramètres : aucun
Retourne dynamicTextCutType.

getFont()

Paramètres : aucun
Retourne la police du texte à afficher.

getFontSize()

Paramètres : aucun
Retourne la taille de police du texte à afficher.

getInput()

Paramètres : aucun
Retourne si le widget est utilisé comme entré ou non.

getText()

Paramètres : aucun
Retourne le texte affiché.

getTextColor()

Paramètres : aucun
Retourne la couleur du texte.

getTextHorizontalAlignment()

Paramètres : aucun
Retourne l'alignement horizontal du texte.

getTextVerticalAlignment()

Paramètres : aucun
Retourne l'alignement vertical du texte.

Paramètres : cursorPosition (nouvelle position du curseur)
Change la valeur de la position du curseur par "cursorPosition".

setCursorVisible()

paramètres : cursorVisible (nouvelle valeur de si le curseur est visible)
Change la valeur de si le curseur est visible par "cursorVisible".

Paramètres : cursorWidth (largeur du curseur en pixel)
Change la largeur du curseur en pixel par "cursorWidth".

setDynamicTextCut()

Paramètres : aucun
Change la valeur de dynamicTextCut à "dynamicTextCut".

setDynamicTextCutType()

Paramètres : dynamicTextCutType (nouvelle valeur de dynamicTextCutType)

setFontSize()

Paramètres : fontSize (nouvelle taille de police)

Paramètres : text (nouveau texte à afficher)

setTextColor()

Paramètres : textColor (nouvelle couleur du texte)

Paramètres : textHorizontalAlignment (nouvel alignement horizontal du texte)

setTextVerticalAlignment()

Paramètres : textVerticalAlignment (nouvel alignement vertical du texte)

appendText()

Paramètres : text (text à ajouter), appendAtCursor (booléen qui indique si le texte est ajouté au niveau du curseur ou pas, basiquement à True), moveCursor (booléen qui indique si le curseur doit bouger avec le texte, basiquement à True)
Ajoute le "text" à text, soit à la fin si appendAtCursor est False, soit au niveau du curseur si appendAtCursor est False, et bouge le curseur si mvoeCursor est True.

_getCursorIsVisible()

Paramètres : aucun
Retourne la valeur de _cursorIsVisible.

_isGettingMouseDown()

Paramètres : button (nombre entier du bouton de souris clické)
Fonction appelé lorsque le widget est clické.
Si button est "1", la touche clické est la touche gauche, si button est "2", la touche clické est la molette, si button est "3", la touche clické est la touche droite.
Ordonne une modification du widget lors d'un click pour commencer à animer le curseur si nécessaire et si la touche pressée est le click gauche.
Cette fonction est héritée de MWidget.

_isKeyGettingDropped()

Paramètres : key (touche lachée)
Fonction appelé lorsque le widget est focus est qu'une touche est pressée.
Reset _backspacePressed et _returnPressed.
Cette fonction est héritée de MWidget.

_isKeyGettingPressed()

Paramètres : key (touche pressée)
Fonction appelé lorsque le widget est focus est qu'une touche est pressée.
Ajoute la touche key dans le texte si input est True.
Cette fonction est utilisé à la place de

_isTextGettingEntered()

pour certains caractères spéciaux (comme le retour à la ligne).
Elle permet aussi de faire naviguer le curseur avec les flèches.
Cette fonction est héritée de MWidget.

_isNotFocusedAnymore()

Paramètres : aucun
Fonction appelé lorsque le widget n'est plus focus.
Permet de savoir quand le widget n'est plus focus pour reset backspacePressed et returnPressed.
Cette fonction est héritée de MWidget.

_isTextGettingEntered()

Paramètres : text (texte entrée)
Fonction appelé lorsque le widget est focus est que l'utilisateur rentre du texte.
Ajoute le texte dans le texte si input est True.
Cette fonction est héritée de MWidget.

_renderBeforeHierarchy()

Paramètres : surface (surface parent)
Met la surface "surface" à jour avec le widget dessus et retourne "surface" mis à jour.
Dans MText, elle s'occupe du texte à afficher. L'ordre précis est :
-Découpe des lignes à afficher (avec ou sans dynamicTextCut)
-Rendu du texte
Cette fonction est héritée de MWidget.

_setCursorIsVisible()

Paramètres : _cursorIsVisible (nouvel valeur de _cursorIsVisible)
Change la valeur de _cursorIsVisible par "_cursorIsVisible".

_update()

Paramètres : deltaTime (temps passé entre cette frame et la dernière frame)
Fonction appelée à chaque frame.
Elle met _backspacePressedTime et _returnPressedTime à jour et d'ajouter les backspace/retour (avec un premier délai de 0.5 secondes et pleins de délais de 0.05 secondes).
Elle fait de même avec leftArrowPressedTime et _rightArrowPressedTime.
Elle s'occupe aussi de faire clignoter le curseur.

Variables

cursorPosition

Position (index du caractère d'avant + 1) du curseur dans le texte.

cursorVisible

Booléen qui indique si le curseur est visible ou non.

cursorWidth

Largeur en pixel du curseur.

dynamicTextCut

Booléen qui indique si le texte doit être coupé en ligne si il est trop long.

dynamicTextCutType

Type de découpe si dynamicTextCut est True.
Si dynamicTextCutType est à 0 : la découpe se fait caractère par caractère.
Si dynamicTextCutType est à 1 : la découpe se fait mot par mot.

font

Police du text à afficher.

fontSize

Taille de police du text à afficher.

input

Booléen qui indique si le MText est utilisé comme une entrée ou non.
Si il est égal à True, alors le MText réagira aux entrées du clavier si il est focus grâce à _isKeyGettingPressed() de MWidget.

text

Texte affiché par le widget.

textColor

Couleur du texte à afficher.

textHorizontalAlignment

Nombre qui correspond à l'alignement horizontale du texte.
Si il est égal à 0, le texte est aligné vers la gauche.
Si il est égal à 1, le texte est aligné vers le milieu.
Si il est égal à 2, le texte est aligné vers la droite.

textVerticalAlignment

Nombre qui correspond à l'alignement verticale du texte.
Si il est égal à 0, le texte est aligné vers le haut.
Si il est égal à 1, le texte est aligné vers le milieu.
Si il est égal à 2, le texte est aligné vers le bas.

_backspacePressed

Booléen qui indique si la touche backspace est pressée ou non.
Elle sert à créer un effet similaire à l'effet de quand on appuie longuement sur un touche et que son action est propagée avec la touche backspace.

_backspacePressedTime

Temps en seconde depuis que _backspacePressed est True.

_backspaceNumber

Nombre de "backspace" appliqués au texte.

_cursorFlashingTime

Temps en seconde d'un clignement de curseur.

_cursorIsVisible

Booléen qui indique si le curseur qui est visible et qui clignote est en mode visible ou non.

_cursorVisibleTime

Temps en seconde depuis que le curseur est devenu visible/invisible.

_leftArrowPressed

Booléen qui indique si la touche flèche gauche est pressée ou non.
Elle sert à créer un effet similaire à l'effet de quand on appuie longuement sur un touche et que son action est propagée avec la touche flèche gauche.

_leftArrowPressedTime

Temps en seconde depuis que _leftArrowPressed est True.

_leftArrowNumber

Nombre de "leftArrow" appliqués au texte.

_returnPressed

Booléen qui indique si la touche retourn est pressée ou non.
Elle sert à créer un effet similaire à l'effet de quand on appuie longuement sur un touche et que son action est propagée avec la touche retour.

_returnPressedTime

Temps en seconde depuis que _returnPressed est True.

_returnNumber

Nombre de "retour" appliqués au texte.

_rightArrowPressed

Booléen qui indique si la touche flèche droite est pressée ou non.
Elle sert à créer un effet similaire à l'effet de quand on appuie longuement sur un touche et que son action est propagée avec la touche flèche droite.

_rightArrowPressedTime

Temps en seconde depuis que _rightArrowPressed est True.

_rightArrowNumber

Nombre de "rightArrow" appliqués au texte.
