MText

MText est une classe permettant de d'afficher du texte.
Elle hérite de MFrame.
Ajouté à la version 1.0.0.

Fonctions

__init__()

Paramètres : text (assignée à la variable text), x (assignée à la variabe x), y (assignée à la variabe y), width (assignée à la variable width), height (assignée à la variable height), parent (assignée à la variable parent), widgetType (assignée à la variable _type, basiquememnt mit à "MText")
Construit un objet MText.
Modifié à la version 1.2.0.

appendText()

Paramètres : text (text à ajouter), appendAtCursor (booléen qui indique si le texte est ajouté au niveau du curseur ou pas, basiquement à True), moveCursor (booléen qui indique si le curseur doit bouger avec le texte, basiquement à True)
Ajoute le "text" à text, soit à la fin si appendAtCursor est False, soit au niveau du curseur si appendAtCursor est False, et bouge le curseur si moveCursor est True.
Ajouté à la version 1.3.0.

getTextRendered()

Paramètres : generator (générateur de texte), pieces (texte à générer, basiquement à 0)
Retourne une liste qui contient les surfaces du texte à générer.
Elle s'occupe aussi d'appliquer les modifications graphiques dû à la selection.
Si pieces est à 0, pieces sera égal à getCuttedPiece.
Modifié à la version 1.3.0.

_isGettingMouseDown()

Paramètres : button (nombre entier du bouton de souris clické), relativePos (position relative au widget du curseur de la souris)
Fonction appelé lorsque le widget est clické.
Si button est "1", la touche clické est la touche gauche, si button est "2", la touche clické est la molette, si button est "3", la touche clické est la touche droite.
Ordonne une modification du widget lors d'un click pour commencer à animer le curseur si nécessaire et si la touche pressée est le click gauche.
Permet aussi de déplacer le curseur au niveau du click et d'annuler la valeur de selectionStart et selectionStop en leur attribuant la même valeur.
Cette fonction est héritée de MWidget.
Modifié à la version 1.3.0.

_isGettingMouseUp()

Paramètres : button (nombre entier du bouton de souris clické), relativePos (position relative au widget du curseur de la souris)
Fonction appelé lorsque le widget n'est plus clické.
Si button est "1", la touche clické est la touche gauche, si button est "2", la touche clické est la molette, si button est "3", la touche clické est la touche droite.
Permet de placer là où la sélection s'arrête.
Cette fonction est héritée de MWidget.
Modifié à la version 1.3.0.

_isKeyGettingDropped()

Paramètres : key (touche lachée)
Fonction appelé lorsque le widget est focus est qu'une touche est pressée.
Reset _backspacePressed et _returnPressed.
Cette fonction est héritée de MWidget.
Ajouté à la version 1.3.0.

_isKeyGettingPressed()

Paramètres : key (touche pressée)
Fonction appelé lorsque le widget est focus est qu'une touche est pressée.
Ajoute la touche key dans le texte si input est True.
Cette fonction est utilisé à la place de

_isTextGettingEntered()

pour certains caractères spéciaux (comme le retour à la ligne).
Elle permet aussi de faire naviguer le curseur avec les flèches.
Cette fonction est héritée de MWidget.
Ajouté à la version 1.3.0.

_isNotFocusedAnymore()

Paramètres : aucun
Fonction appelé lorsque le widget n'est plus focus.
Permet de savoir quand le widget n'est plus focus pour reset backspacePressed, returnPressed, les flèches et la touche contrôle.
Cette fonction est héritée de MWidget.
Ajouté à la version 1.3.0.

_isTextGettingEntered()

Paramètres : text (texte entrée)
Fonction appelé lorsque le widget est focus est que l'utilisateur rentre du texte.
Ajoute le texte dans le texte si input est True.
Cette fonction est héritée de MWidget.
Ajouté à la version 1.0.0.

_removeTextAtCursor()

Paramètres : length (taille du texte à supprimer)
Supprime un texte de longeur "length" derrière le curseur.
Ajouté à la version 1.0.0.

_renderBeforeHierarchy()

Paramètres : surface (surface parent)
Met la surface "surface" à jour avec le widget dessus et retourne "surface" mis à jour.
Dans MText, elle s'occupe du texte à afficher. L'ordre précis est :
-Découpe des lignes à afficher (avec ou sans dynamicTextCut)
-Rendu du texte
Cette fonction est héritée de MWidget.
Modifié à la version 1.3.0.

_setCursorIsVisible()

Paramètres : _cursorIsVisible (nouvel valeur de _cursorIsVisible)
Change la valeur de _cursorIsVisible par "_cursorIsVisible".
Ajouté à la version 1.0.0.

_update()

Paramètres : deltaTime (temps passé entre cette frame et la dernière frame)
Fonction appelée à chaque frame.
Elle met _backspacePressedTime et _returnPressedTime à jour et d'ajouter les backspace/retour (avec un premier délai de 0.5 secondes et pleins de délais de 0.05 secondes).
Elle fait de même avec leftArrowPressedTime et _rightArrowPressedTime.
Elle s'occupe aussi de faire clignoter le curseur.
Ajouté à la version 1.0.0.

Variables

cursorPosition

Position (index du caractère d'avant + 1) du curseur dans le texte.
Ajouté à la version 1.0.0.

cursorVisible

Booléen qui indique si le curseur est visible ou non.
Ajouté à la version 1.0.0.

cursorWidth

Largeur en pixel du curseur.
Ajouté à la version 1.0.0.

dynamicTextCut

Booléen qui indique si le texte doit être coupé en ligne si il est trop long.
Ajouté à la version 1.0.0.

dynamicTextCutType

Type de découpe si dynamicTextCut est True.
Si dynamicTextCutType est à 0 : la découpe se fait caractère par caractère.
Si dynamicTextCutType est à 1 : la découpe se fait mot par mot.
Ajouté à la version 1.0.0.

font

Police du text à afficher.
Ajouté à la version 1.0.0.

fontSize

Taille de police du text à afficher.
Ajouté à la version 1.0.0.

input

Booléen qui indique si le MText est utilisé comme une entrée ou non.
Si il est égal à True, alors le MText réagira aux entrées du clavier si il est focus grâce à _isKeyGettingPressed() de MWidget.
Ajouté à la version 1.0.0.

selection

Indique si la sélection dans le texte est possible ou non.
Ajouté à la version 1.3.0.

selectionBackgroundColor

Couleur derrière la partie sélectionnée du texte.
Ajouté à la version 1.3.0.

selectionStart

Position dans le texte où la sélection commence.
Elle utilise le même système de numérotation de la position que cursorPosition.
Si elle est égal à selectionStop, il n'y a pas de sélection.
Ajouté à la version 1.3.0.

selectionStop

Position dans le texte où la sélection s'arrête.
Elle utilise le même système de numérotation de la position que cursorPosition.
Si elle est égal à selectionStart, il n'y a pas de sélection.
Ajouté à la version 1.3.0.

selectionTextColor

Couleur du texte de la partie sélectionnée.
Ajouté à la version 1.3.0.

text

Texte affiché par le widget.
Ajouté à la version 1.0.0.

textBottomOffset

Décalage du texte en bas (avec la bordure basse de MFrame). Ajouté à la version 1.2.0.

textColor

Couleur du texte à afficher.
Ajouté à la version 1.0.0.

textLeftOffset

Décalage du texte à gauche (avec la bordure gauche de MFrame). Ajouté à la version 1.2.0.

textRightOffset

Décalage du texte à droite (avec la bordure droite de MFrame). Ajouté à la version 1.2.0.

textHorizontalAlignment

Nombre qui correspond à l'alignement horizontale du texte.
Si il est égal à 0, le texte est aligné vers la gauche.
Si il est égal à 1, le texte est aligné vers le milieu.
Si il est égal à 2, le texte est aligné vers la droite.
Ajouté à la version 1.0.0.

textTopOffset

Décalage du texte en haut (avec la bordure haute de MFrame). Ajouté à la version 1.2.0.

textVerticalAlignment

Nombre qui correspond à l'alignement verticale du texte.
Si il est égal à 0, le texte est aligné vers le haut.
Si il est égal à 1, le texte est aligné vers le milieu.
Si il est égal à 2, le texte est aligné vers le bas.
Ajouté à la version 1.0.0.
