MBar

MBar est une classe secondaire pour définir une barre de progression.
Elle hérite de MFrame.
Ajouté à la version 2.3.0.

Fonctions

__init__()

Paramètres : x (assignée à la variabe x de MWidget), y (assignée à la variabe y de MWidget), width (assignée à la variable width de MWidget), height (assignée à la variable height de MWidget), parent (assignée à la variable parent de MWidget), widgetType (assignée à la variable _type de MWidget, basiquememnt mit à "MBar")
Construit un objet MBar.
Ajouté à la version 2.3.0.

barRect()

Paramètres : aucun
Retourne le tuple (x, y, width, height) de la barre de progression.
Ajouté à la version 2.3.0.

barSurface()

Paramètres : aucun
Retourne la surface de la barre de progression du MBar.
Ajouté à la version 2.3.0.

getAnimation()

Paramètres : aucun
Retourne si l'animation est jouée.
Ajouté à la version 2.3.0.

getAnimationColor()

Paramètres : aucun
Retourne la couleur de l'animation.
Ajouté à la version 2.3.0.

getAnimationSpeed()

Paramètres : aucun
Retourne la vitesse de l'animation.
Ajouté à la version 2.3.0.

getBarColor()

Paramètres : aucun
Retourne la couleur de la barre.
Ajouté à la version 2.3.0.

getBarLengthInOrientationAxis()

Paramètres : aucun
Retourne la largeur ou la hauteur de la barre de progression selon l'orientation.
Ajouté à la version 2.3.0.

getMaxValue()

Paramètres : aucun
Retour la valeur maximale stockée dans le MBar.
Ajouté à la version 2.3.0.

getMinValue()

Paramètres : aucun
Retour la valeur minimale stockée dans le MBar.
Ajouté à la version 2.3.0.

getRange()

Paramètres : aucun
Retourne maxValue - minValue.
Ajouté à la version 2.3.0.

getOrientation()

Paramètres : aucun
Retourne l'orientation de progession du MBar.
Ajouté à la version 2.3.0.

getOrientationAxisLength()

Paramètres : aucun
Retourne height si l'orientation est ORIENTATION_BOTTOM_TO_TOP ou ORIENTATION_TOP_TO_BOTTOM, sinon retourne width.
Ajouté à la version 2.3.0.

getValue()

Paramètres : aucun
Retour la valeur stockée dans le MBar.
Ajouté à la version 2.3.0.

setAnimation()

Paramètres : animation (nouvelle valeur de si l'animation est jouée)
Change si l'animation est jouée par "animation".
Ajouté à la version 2.3.0.

setAnimationColor()

Paramètres : animationColor (nouvelle couleur de l'animation)
Change la couleur de l'animation par "animationColor".
Ajouté à la version 2.3.0.

setAnimationSpeed()

Paramètres : animationSpeed (nouvelle vitesse de l'animation)
Change la vitesse de l'animation par "animationSpeed".
Ajouté à la version 2.3.0.

setBarColor()

Paramètres : barColor (nouvelle couleur de la barre)
Change la couleur de la barre par barColor.
Ajouté à la version 2.3.0.

setMaxValue()

Paramètres : maxValue (nouvelle valeur maximale stockée dans le MBar)
Change la valeur maximale stockée dans le MBar par "maxValue".
Ajouté à la version 2.3.0.

setMinValue()

Paramètres : minValue (nouvelle valeur minimale stockée dans le MBar)
Change la valeur minimale stockée dans le MBar par "minValue".
Ajouté à la version 2.3.0.

setValue()

Paramètres : value (nouvelle valeur stockée dans le MBar)
Change la valeur stockée dans le MBar par "value".
Ajouté à la version 2.3.0.

setWidth()

Paramètres : width (nouvelle largeur du widget)
Change la largeur du widget pour newWidth.
Modifie aussi _animationSize si nécessaire.
Cette fonction est héritée de MWidget.
Ajouté à la version 2.3.0.

_getAnimationState()

Paramètres : aucun
Retourne l'état de l'animation.
Ajouté à la version 2.3.0.

_renderBeforeHierarchy()

Paramètres : surface (surface parent)
Met la surface "surface" à jour avec le widget dessus et retourne "surface" mis à jour.
Dans MBar, elle s'occupe d'afficher la barre de progression.
Cette fonction est héritée de MWidget.
Modifié à la version 2.3.0.

_update()

Paramètres : deltaTime (temps passé entre cette frame et la dernière frame)
Fonction appelée à chaque frame.
Elle permet de gérer l'animation.
Ajouté à la version 2.3.0.

Variables statiques

ORIENTATION_BOTTOM_TO_TOP

Valeur de ORIENTATION si la barre d'orientation va de bas en haut, donc 3.
Ajouté à la version 2.3.0.

ORIENTATION_LEFT_TO_RIGHT

Valeur de ORIENTATION si la barre d'orientation va de la gauche à droite, donc 0.
Ajouté à la version 2.3.0.

ORIENTATION_RIGHT_TO_LEFT

Valeur de ORIENTATION si la barre d'orientation va de la droite à gauche, donc 2.
Ajouté à la version 2.3.0.

ORIENTATION_TOP_TO_BOTTOM

Valeur de ORIENTATION si la barre d'orientation va de haut en bas, donc 1.
Ajouté à la version 2.3.0.

Variables

animation

Booléen qui indique si une animation est jouée.
Ajouté à la version 2.3.0.

animationSpeed

Vitesse de l'animation.
Ajouté à la version 2.3.0.

barColor

Couleur de la barre de progression.
Ajouté à la version 2.3.0.

maxValue

Valeur maximale stockée dans le MBar.
Ajouté à la version 2.3.0.

minValue

Valeur minimale stockée dans le MBar.
Ajouté à la version 2.3.0.

ORIENTATION

Orientation du MBar vers laquelle progresse la barre de progression.
Si il est égal à ORIENTATION_BOTTOM_TO_TOP, la barre d'orientation va de bas en haut.
Si il est égal à ORIENTATION_LEFT_TO_RIGHT, la barre d'orientation va de la gauche à droite.
Si il est égal à ORIENTATION_RIGHT_TO_LEFT, la barre d'orientation va de la droite à gauche.
Si il est égal à ORIENTATION_TOP_TO_BOTTOM, la barre d'orientation va de haut en bas.
Ajouté à la version 2.3.0.

value

Valeur stockée dans le MBar.
Ajouté à la version 2.3.0.

_animationSize

Taille de l'animation.
Il est conseillé de le mettre à la taille de l'axe d'orientation divisé par 10.
Ajouté à la version 2.3.0.

animationState

État de l'animation (entre 0 et la taille de la surface sur l'axe d'orientation).
Ajouté à la version 2.3.0.
