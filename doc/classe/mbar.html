MBar

MBar est une classe secondaire pour définir une barre/slider.
Elle hérite de MFrame.
Ajouté à la version 2.0.0.

Affichage

MBar hérite de MFrame. Elle contient une petite bordure de 1 de large. Elle peut être horizontale ou verticale selon ORIENTATION. Elle contient un bouton amovible pour changer la valeur dans le MBar.

Fonctions

__init__()

Paramètres : x (assignée à la variabe x de MWidget), y (assignée à la variabe y de MWidget), width (assignée à la variable width de MWidget), height (assignée à la variable height de MWidget), parent (assignée à la variable parent de MWidget), widgetType (assignée à la variable _type de MWidget, basiquememnt mit à "MBar")
Construit un objet MBar.
Ajouté à la version 2.0.0.

getButtonBackgroundColor()

Paramètres : aucun
Retourne la valeur de la couleur d'arrière plan du bouton amovible.
Ajouté à la version 2.0.0.

getButtonBackgroundColorOnOverflight()

Paramètres : aucun
Retourne la valeur de la couleur d'arrière plan du bouton amovible lors du survol du curseur.
Ajouté à la version 2.0.0.

getButtonOrientationLength

Paramètres : aucun

getChangeButtonBackgroundColorOnOverflight()

Paramètres : aucun
Retourne le booléen qui indique si la couleur d'arrière plan du bouton amovible change au survol du curseur.
Ajouté à la version 2.0.0.

getMaxValue()

Paramètres : aucun
Retourne la valeur maximale possible dans le MBar.
Ajouté à la version 2.0.0.

getMinValue()

Paramètres : aucun
Retourne la valeur minimale possible dans le MBar.
Ajouté à la version 2.0.0.

getStep()

Paramètres : aucun
Retourne le pas entre chaque valeur dans le MBar.
Ajouté à la version 2.0.0.

getValue()

Paramètres : aucun
Retourne la valeur actuelle dans le MBar.
Ajouté à la version 2.0.0.

getButtonOrientationPos()

Paramètres : aucun
Calcul et renvoie la coordonnée sur l'axe de navigation du bouton selon value.
Ajouté à la version 2.0.0.

isValueIn()

Paramètres : value (valeur à tester)
Retourne si "value" est entre minValue et maxValue.
Ajouté à la version 2.0.0.

setButtonBackgroundColor()

Paramètres : buttonBackgroundColor (nouvelle valeur de la couleur d'arrière plan du bouton amovible)
Change la valeur de la couleur d'arrière plan du bouton amovible par "buttonBackgroundColor".
Ajouté à la version 2.0.0.

setButtonBackgroundColorOnOverflight()

Paramètres : buttonBackgroundColorOnOverflight (nouvelle valeur de la couleur d'arrière plan du bouton amovible lors du survol du curseur)
Change la valeur de la couleur d'arrière plan du bouton amovible lors du survol du curseur par "buttonBackgroundColorOnOverflight".
Ajouté à la version 2.0.0.

setChangeButtonBackgroundColorOnOverflight()

Paramètres : changeButtonBackgroundColorOnOverflight (nouvelle valeur du booléen qui indique si la couleur d'arrière plan du bouton amovible change au survol du curseur)
Retourne le booléen qui indique si la couleur d'arrière plan du bouton amovible change au survol du curseur par "changeButtonBackgroundColorOnOverflight".
Ajouté à la version 2.0.0.

setMaxValue()

Paramètres : maxValue (nouvelle valeur maximale possible dans le MBar)
Change la valeur maximale possible dans le MBar par "maxValue".
Cette valeur doit être supérieur ou égal à minValue et value.
Ajouté à la version 2.0.0.

setMinValue()

Paramètres : minValue (nouvelle valeur minimale possible dans le MBar)
Change la valeur minimale possible dans le MBar par "minValue".
Cette valeur doit être sinférieur ou égal à maxValue et value.
Ajouté à la version 2.0.0.

setStep()

Paramètres : step (nouveau pas entre chaque valeur dans le MBar)
Change le pas entre chaque valeur dans le MBar par "step".
Ajouté à la version 2.0.0.

setValue()

Paramètres : value (nouvelle valeur actuelle dans le MBar)
Change la valeur actuelle dans le MBar par "value".
Ajouté à la version 2.0.0.

_doNotOverflightedEffect()

Paramètres : aucun
Effectue sur le MBar les actions nécessaires lorsque le MBar est normal (non survolé).
Ajouté à la version 2.0.0.

_doOverflightedEffect()

Paramètres : aucun
Effectue sur le MBar les actions nécessaires lorsque le MBar est survolé.
Ajouté à la version 2.0.0.

_getValueAtPos()

Paramètres : relativePos (position à tester dans le MBar)
Retourne la valeur du MBar à la position "relativePos".
Ajouté à la version 2.0.0.

_isGettingMouseDown()

Paramètres : button (nombre entier du bouton de souris clické), relativePos (position relative au widget du curseur de la souris)
Fonction appelé lorsque le widget est clické.
Si button est "1", la touche clické est la touche gauche, si button est "2", la touche clické est la molette, si button est "3", la touche clické est la touche droite.
Met à jour _buttonClicked si nécessaire.
Cette fonction est héritée de MWidget.
Ajouté à la version 2.0.0.

_isGettingMouseUp()

Paramètres : button (nombre entier du bouton de souris clické), relativePos (position relative au widget du curseur de la souris)
Fonction appelé lorsque le widget n'est plus clické.
Si button est "1", la touche clické est la touche gauche, si button est "2", la touche clické est la molette, si button est "3", la touche clické est la touche droite.
Met à jour _buttonClicked à "False".
Cette fonction est héritée de MWidget.
Ajouté à la version 2.0.0.

_isGettingOverflighted()

Paramètres : aucun
Fonction appelé lorsque le widget est survolé.
Permet d'appeler _doOverflightedEffect lors du survol du curseur.
Cette fonction est héritée de MWidget.
Ajouté à la version 2.0.0.

_isNotOverflightedAnymore()

Paramètres : aucun
Fonction appelé lorsque le widget n'est plus survolé par le curseur.
Permet de savoir quand le widget n'est plus survolé pour appeler _doNotOverflightedEffect.
Cette fonction est héritée de MWidget.
Ajouté à la version 2.0.0.

_isPosOverButton()

Paramètres : relativePos (position à tester)
Retourne si la position "relativePos" est au dessus du bouton amovible.
Ajouté à la version 2.0.0.

_renderBeforeHierarchy()

Paramètres : surface (surface parent)
Met la surface "surface" à jour avec le widget dessus et retourne "surface" mis à jour.
Dans MBar, elle s'occupe d'afficher le bouton amovible.
Cette fonction est héritée de MWidget.
Modifié à la version 2.0.0.

_buttonOrientationLength

Longueur du bouton amovible sur l'axe de l'orientation.

changeButtonBackgroundColorOnOverflight

Booléen qui indique si le bouton amovible est clické ou non.
Ajouté à la version 2.0.0.

_buttonOverflighted

Booléen qui indique si le bouton amovible est survolé par le curseur ou non.
Ajouté à la version 2.0.0.
