MChrono

MChrono est une classe secondaire implicite pour définir une liste de bouton cochable.
Elle hérite de MObject.
Ajouté à la version 2.3.0.

Fonctions

__init__()

Paramètres : format (assignée à la variable format), x (assignée à la variabe x), y (assignée à la variabe y), width (assignée à la variable width), height (assignée à la variable height), parent (assignée à la variable parent), widgetType (assignée à la variable _type, basiquememnt mit à "MChrono")
Construit un objet MChrono.
Ajouté à la version 2.3.0.

addNanoSecond()

Paramètres : ns (nombre de nano secondes à ajouter)
Rajoute "ns" nano secondes au chronomètre.
Ajouté à la version 2.3.0.

addSecond()

Paramètres : second (nombre de secondes à ajouter)
Rajoute "second" secondes au chronomètre.
Ajouté à la version 2.3.0.

getChronometerWithFormat()

Paramètres : aucun
Retourne le temps depuis que le chronomètre tourne selon le format du MChrono.
Ajouté à la version 2.3.0.

getFormat()

Paramètres : aucun
Retourne le format du MChrono.
Ajouté à la version 2.3.0.

getTimeSinceStart()

Paramètres : ns (si le retour doit être en nano seconde ou pas)
Retourne le temps depuis le premier démarrage du chronomètre, en nano seconde si ns est True (seconde sinon).
Ajouté à la version 2.3.0.

getUnitSeparation()

Pramètres : aucun
Retourne ce qui sépare les unités dans le format.
Ajouté à la version 2.3.0.

isStarted()

Paramètres : aucun
Retourne si le chronomètre est en marche ou non.
Ajouté à la version 2.3.0.

reset()

Paramètres : aucun
Remet le chronomètre à 0 en vidant importantTime.
Ajouté à la version 2.3.0.

setFormat()

Paramètres : format (nouveau format du MChrono)
Change le format du MChrono par "format".
Ajouté à la version 2.3.0.

setSpeed()

Paramètres : speed (nouvelle valeur de la vitesse du chronomètre)
Change la vitesse du MChrono par "speed".
Ajouté à la version 2.3.0.

setUnitSeparation()

Pramètres : untiSeparation (nouvelle séparation des unités dans le format)
Change ce qui sépare les unités dans le format par "unitSeparation".
Ajouté à la version 2.3.0.

star()

Paramètres : offset (temps auquel le chronomètre doit commencer)
Commence (ou recommence) le chronomètre avec un nombre "offset" de nanoseconde d'avance.
Ajouté à la version 2.3.0.

stop()

Paramètres : aucun
Arrête le chronomètre.
Ajouté à la version 2.3.0.

updateChronometer()

Paramètres : aucun
Met à jour le chronomètre avec le bon texte.
Ajouté à la version 2.3.0.

Variables statiques

FORMAT_HH_MM_CS

Format heure_minute_centieme de seconde.
Ajouté à la version 2.3.0.

FORMAT_HH_MM_SS

Format heure_minute_seconde.
Ajouté à la version 2.3.0.

Variables

format

Format d'affichage du chronomètre (il est conseillé d'utiliser les vaiables statiques de la classe).
Ajouté à la version 2.3.0.

importantTime

Liste de temps (en time_ns) selon le pattern start-stop-start-stop-...-start (chronomètre lancé) ou stop (chronomètre arrêté).
Ajouté à la version 2.3.0.

speed

Vitesse du chronomètre (si X2, speed = 2).
Ajouté à la version 2.3.0.

started

Booléen qui indique si le chronomètre est en marche ou non.
Ajouté à la version 2.3.0.

unitSeparation

Chaîne de caractère qui sépare les différentes parties du format (seconde, minute, heure...)
Ajouté à la version 2.3.0.

_nsToAdd

Nombre de nano secondes à ajouter au chrono.
Ajouté à la version 2.3.0.
