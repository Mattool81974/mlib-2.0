MImage

MImage est une classe secondaire permettant de créer des objets graphiques affichant une image.
Elle hérite de MFrame.
Ajouté à la version 1.0.0.

Fonctions

__init__()

Paramètres : imageLink (chemin d'accés vers l'image à afficher), x (assignée à la variabe x), y (assignée à la variabe y), width (assignée à la variable width), height (assignée à la variable height), parent (assignée à la variable parent), widgetType (assignée à la variable _type, basiquememnt mit à "MFrame")
Construit un objet MImage.
Ajouté à la version 1.0.0.

getImageHorizontalAlignment

Paramètres : aucun
Retourne l'alignement horizontal de l'image.
Ajouté à la version 1.1.0.

getImageLink()

Paramètres : aucun
Retourne le chemin d'accés vers l'image à afficher.
Ajouté à la version 1.0.0.

getImagePosition()

Paramètres : aucun
Retourne la position de l'image.
Ajouté à la version 1.1.0.

getImageReframing()

Paramètres : aucun
Retourne le recadrement de l'image.
Ajouté à la version 1.1.0.

getImageSize()

Paramètres : aucun
Retourne la taille de l'image défini pour le recadrement.
Ajouté à la version 1.1.0.

getImageVerticalAlignment()

Paramètres : aucun
Retourne l'alignement vertical de l'image.
Ajouté à la version 1.1.0.

setImageHorizontalAlignment

Paramètres : imageHorizontalAlignment (nouvel alignement horizontal de l'image)
Change l'alignement horizontal de l'image par "imageHorizontalAlignment".
Ajouté à la version 1.1.0.

setImageLink()

Paramètres : imageLink (Nouveau chemin d'accés vers l'image), editSize (booléen qui indique si la taille de l'image selon le recadrement est modifié par la taille de la nouvelle image ou non, basiquement à True)
Change le chemin d'accés vers l'image par "imageLink" si "imageLink" existe dans le système.
Elle change aussi l'image pygame.
Et si "editSize" est True, elle change aussi la taille de l'image selon le recadrement par la taille de la nouvelle image.
Modifié à la version 1.1.0.

setImagePosition()

Paramètres : imagePosition (nouvelle position de l'image)
Change la position de l'image par "imagePosition".
Ajouté à la version 1.1.0.

setImageReframing()

Paramètres : imageReframing (nouveau recadrement de l'image)
Change le recadrement de l'image à "imageReframing".
Ajouté à la version 1.1.0.

setImageSize()

Paramètres : imageSize (nouvelle taille de l'image défini pour le recadrement)
Change la taille de l'image défini pour le recadrement à "imageSize".
Ajouté à la version 1.1.0.

setImageVerticalAlignment()

Paramètres : imageVerticalAlignment (nouvel alignement vertical de l'image)
Change l'alignement vertical de l'image par "imageVerticalAlignment".
Ajouté à la version 1.1.0.

_renderBeforeHierarchy()

Paramètres : surface (surface parent)
Met la surface "surface" à jour avec le widget dessus et retourne "surface" mis à jour.
Dans Mimage, elle affiche l'image sur la surface.
Elle s'occupe de l'alignement/position de l'image, mais pas de la taille (c'est _resizeImage qui s'en occupe).
Cette fonction est héritée de MWidget.
Ajouté à la version 1.1.0.

_resizeImage()

Paramètres : aucun
Recadre l'image selon le imageReframing et stocke le résultat dans _imageToDraw.
Ajouté à la version 1.1.0.

Variables

imageHorizontalAlignment

Déifni l'alignement horizontal de l'image.
Si il est égal à -1, l'image est située à la coordonnée x de imagePosition.
Si il est égal à 0, l'image est alignée à gauche du MImage.
Si il est égal à 1, l'image est alignée au milieu du MImage.
Si il est égal à 2, l'image est alignée à droite du MImage.
Ajouté à la version 1.1.0.

imageLink

Chemin d'accés vers l'image à afficher.
Ajouté à la version 1.0.0.

imagePosition

Position de l'image à afficher.

imageReframing

Défini le recadrement de l'image.
Si il est égal à 0, il n'y a aucun recadrement.
Si il est égal à 1, l'image est recadré pour tenir sur l'axe horizontal dans le MImage.
Si il est égal à 2, l'image est recadré pour tenir sur l'axe vertical dans le MImage.
Si il est égal à 3, l'image est recadré pour tenir entièrement dans l'image dans le MImage.
Si il est égal à 4, l'image est recadré selon imageSize.
Ajouté à la version 1.1.0.

imageSize

Défini la taille de l'image si imageReframing est égal à 4.
Ajouté à la version 1.1.0.

imageVerticalAlignment

Déifni l'alignement vertical de l'image.
Si il est égal à -1, l'image est située à la coordonnée y de imagePosition.
Si il est égal à 0, l'image est alignée en haut du MImage.
Si il est égal à 1, l'image est alignée au milieu du MImage.
Si il est égal à 2, l'image est alignée en bas du MImage.
Ajouté à la version 1.1.0.

_image

Objet pygame image de l'image; assigné par setImageLink.
Ajouté à la version 1.1.0.

_imageToDraw

Objet pygame image de l'image, modifié avec le recadrement nécessaire.
Ajouté à la version 1.1.0.
