MApp

MApp est la classe permettant de définir la fenêtre principale de l'interface graphique. Elle hérite de mwidget.
Ajouté à la version 1.0.0.

Fonctions

__init__()

Paramètres : pygameWindow (assignée à la variable pygameWindow), windowTitle (assignée à la variable windowTitle), windowWidth (défini la largeur de la fenêtre), windowHeight (défini la hauteur de la fenêtre), printFps (assignée à la variable printFps)
Crée la fenêtre sur lequel effectuer les modifications graphiques.
Ajouté à la version 1.0.0.

frame()

Paramètres : Aucun
Effectue toutes les actions de la fenêtre graphique.
Ajouté à la version 1.0.0.

frameEvent()

Paramètres : Aucun
Effectue toutes les actions de la fenêtre graphique qui correspondent aux évènements.
Cette fonction s'occupe aussi d'actualiser le deltaTime puis fps.
Il est FORTEMENT conseillé d'appeler cette fonction au début d'une frame.
Modifié à la version 1.3.0.

frameGraphics()

Paramètres : Aucun
Effectue toutes les actions de la fenêtre graphique qui correspondent aux graphismes.
Il est FORTEMENT conseillé d'appeler cette fonction à la fin d'une frame.
Ajouté à la version 1.0.0.

getDeltaTime()

Paramètres : aucun
Retourne deltaTime.
Ajouté à la version 1.0.0.

getFps()

Paramètres : aucun
Retourne fps.
Ajouté à la version 1.0.0.

getPressedKey()

Paramètres : aucun
Retourne pressedKey.
Ajouté à la version 1.0.0.

getPrintFps()

Paramètres : aucun
Retourne printFps.
Ajouté à la version 1.0.0.

getWidgets()

Paramètres : aucun
Retourne _widgets.
Ajouté à la version 1.0.0.

isKeyPressed()

Paramètres : key (touche à vérifier)
Retourne si la touche "key" est pressé (si elle est dans pressedKey).
Ajouté à la version 1.0.0.

setPrintFps()

Paramètres : printFps (nouvelle valeur de printFps)
Change la valeur de printFps.
Ajouté à la version 1.0.0.

setWindowTitle()

Paramètres : windowTitle (nouveau titre de la fenêtre)
Change le titre de la fenêtre.
Ajouté à la version 1.0.0.

_containsWidget

Paramètres : widget (widget à chercher)
Retourne si le "widget" est contenu dans _widgets.
Ajouté à la version 1.0.0.

_declaringWidget()

Paramètres : widget (widget à enregistrer)
Fonction permettant d'enregistrer un widget nouvellement crée dans _widgets.
Ajouté à la version 1.0.0.

Variables

cursorOnOnverflight

Même variable que pour le mwidget.
La particularité de ce curseur est que celui là représente le curseur de base de l'interface graphique.
Ajouté à la version 1.0.0.

deltaTime

Temps entre la dernière frame évènement et la frame évènement actuel.
Ajouté à la version 1.0.0.

focusedWidget

Widget focus par l'utilisateur (quand l'utilisateur click dessus).
Ajouté à la version 1.0.0.

fps

Nombre de frames par secondes.
Ajouté à la version 1.0.0.

pressedKey

Liste des touches pressées pendant la frame.
Ajouté à la version 1.0.0.

fps

Booléen qui détermine si le MApp print le nombre de FPS à chaque updates de fps.
Ajouté à la version 1.0.0.

windowTitle

Texte affiché en haut de la fenêtre graphique.
Ajouté à la version 1.0.0.

deltaTime

Variable cacher pour permettre d'actualiser deltaTime.
Ajouté à la version 1.0.0.

fpsCount

Nombre de frames depuis la dernière update de fps.
Ajouté à la version 1.0.0.

fpsDuration

Nombre de secondes depuis la dernière update de fps.
Ajouté à la version 1.0.0.

_modifiedWidget

Liste de tous les widgets modifiés à la dernière frame dans frameEvent pour leur appliquer resetWidget().
Ajouté à la version 1.0.0.

pygameWindow

Fenêtre pygame de base ou effectuer les modifications.
Ajouté à la version 1.0.0.

_widgets

Liste de tous les widgets présents dans le MApp.
Ajouté à la version 1.0.0.
