MApp

MApp est la classe primaire permettant de définir la fenêtre principale de l'interface graphique.
Elle hérite de mwidget.
Ajouté à la version 1.0.0.

Système de console

MApp contient un système de console qui permet de garder un oeil sur le déroulé du programme. Des l'entrée dans une fonction, une petite ligne de code est ajoutée pour l'enregistrer dans la console. La console est écrit une fois par seconde dans un fichier au chemin d'accés consoleFile une fois par seconde. Toutes les fonctions modifiées par un ajout console ne seront pas marquées comme modifiées à la version actuelle.

Fonctions

__init__()

Paramètres : pygameWindow (assignée à la variable pygameWindow), windowTitle (assignée à la variable windowTitle), windowWidth (défini la largeur de la fenêtre), windowHeight (défini la hauteur de la fenêtre), printFps (assignée à la variable printFps), windowIcon (chemin d'accés vers l'icône de l'application)
Crée la fenêtre sur lequel effectuer les modifications graphiques.
Ajouté à la version 1.0.0.

frame()

Paramètres : Aucun
Effectue toutes les actions de la fenêtre graphique.
Ajouté à la version 1.0.0.

frameEvent()

Paramètres : Aucun
Effectue toutes les actions de la fenêtre graphique qui correspondent aux évènements.
Cette fonction s'occupe aussi d'actualiser le deltaTime, puis fps, puis de stocker la console sous format ".txt".
Il est FORTEMENT conseillé d'appeler cette fonction au début d'une frame.
Modifié à la version 2.0.1.

frameGraphics()

Paramètres : Aucun
Effectue toutes les actions de la fenêtre graphique qui correspondent aux graphismes.
Juste avant, elle appelle _lastUpdate de tous les widgets.
Il est FORTEMENT conseillé d'appeler cette fonction à la fin d'une frame.
Modifié à la version 2.0.1.

getConsole()

Paramètres : aucun
Retourne si la console est activée.
Ajouté à la version 1.4.0.

getConsoleContent()

Paramètres : aucun
Retourne la valeur du contenu de la console.
Ajouté à la version 1.4.0.

getConsoleFile()

Paramètres : aucun
Retourne la valeur du chemin d'accés du fichier où stocker la console.
Ajouté à la version 1.4.0.

getDeltaTime()

Paramètres : aucun
Retourne deltaTime.
Ajouté à la version 1.0.0.

getFps()

Paramètres : aucun
Retourne fps.
Ajouté à la version 1.0.0.

getPressedKey()

Paramètres : aucun
Retourne pressedKey.
Ajouté à la version 1.0.0.

getPrintFps()

Paramètres : aucun
Retourne printFps.
Ajouté à la version 1.0.0.

getWidgets()

Paramètres : aucun
Retourne _widgets.
Ajouté à la version 1.0.0.

getWindowIcon()

Paramètres : aucun
Retourne la valeur du chemin d'accés vers l'icône de la fenêtre.
Ajouté à la version 1.4.0.

getWindowTitle()

Paramètres : aucun
Retourne le titre de la fenêtre.
Ajouté à la version 1.4.0.

isKeyPressed()

Paramètres : key (touche à vérifier)
Retourne si la touche "key" est pressé (si elle est dans pressedKey).
Ajouté à la version 1.0.0.

setConsole()

Paramètres : console (nouvelle valeur de si la console est active ou non)
Change la valeur de si la console est active ou non par "console".
Ajouté à la version 1.4.0.

setConsoleContent()

Paramètres : consoleContent (contenu du texte de la console)
Change la valeur du contenu du texte de la console par "consoleContent".
Ajouté à la version 1.4.0.

setConsoleFile()

Paramètres : consoleFile (nouveau chemin d'accés du fichier où stocker la console)
Change la valeur du chemin d'accés du fichier où stocker la console par "consoleFile".
Ajouté à la version 1.4.0.

setPrintFps()

Paramètres : printFps (nouvelle valeur de printFps)
Change la valeur de printFps.
Ajouté à la version 1.0.0.

setWindowIcon()

Paramètres : windowIcon (nouveau chemin d'accés vers l'icône de la fenêtre)
Change la valeur du chemin d'accés vers l'icône de la fenêtre par "windowIcon".
Ajouté à la version 1.4.0.

setWindowTitle()

Paramètres : windowTitle (nouveau titre de la fenêtre)
Change le titre de la fenêtre.
Ajouté à la version 1.0.0.

writeConsole()

Paramètres : toWrite (chose à écrire dans la console), indentation (nombre d'espace * 2 à rajouter avant toWrite, basiquement à 0), writer (objet appelant la fonction, basiquement à 0)
Permet d'écrire quelque chose dans la console.
Elle rajoute au texte l'indentation, le type de l'objet, l'ID de l'objet et la date/heure.
Ajouté à la version 1.4.0.

_containsWidget

Paramètres : widget (widget à chercher)
Retourne si le "widget" est contenu dans _widgets.
Ajouté à la version 1.0.0.

_declaringWidget()

Paramètres : widget (widget à enregistrer)
Fonction permettant d'enregistrer un widget nouvellement crée dans _widgets.
Ajouté à la version 1.0.0.

_undeclaringWidget()

Paramètres : widget (widget à désenregistrer)
Fonction permettant de supprimer tout liens avec un widget dans le MApp.
Ajouté à la version 2.0.1.

Variables

console

Booléen qui indique si la console est utilisé ou non.
Ajouté à la version 1.4.0.

consoleContent

Contenu du texte de la console.
Ajouté à la version 1.4.0.

consoleFile

Chemin d'accés du fichier où stocker la console, basiquement à "./console.txt".
Ajouté à la version 1.4.0.

cursorOnOnverflight

Même variable que pour le mwidget.
La particularité de ce curseur est que celui là représente le curseur de base de l'interface graphique.
Ajouté à la version 1.0.0.

deltaTime

Temps entre la dernière frame évènement et la frame évènement actuel.
Ajouté à la version 1.0.0.

frameCount

Nombre de frameEvent depuis le début du programme.
Ajouté à la version 1.4.0.

focusedWidget

Widget focus par l'utilisateur (quand l'utilisateur click dessus).
Ajouté à la version 1.0.0.

fps

Nombre de frames par secondes.
Ajouté à la version 1.0.0.

pressedKey

Liste des touches pressées pendant la frame.
Ajouté à la version 1.0.0.

fps

Booléen qui détermine si le MApp print le nombre de FPS à chaque updates de fps.
Ajouté à la version 1.0.0.

windowIcon

Chemin d'accés vers l'icône de la fenêtre (en haut à gauche de la fenêtre).
Ajouté à la version 1.4.0.

windowTitle

Texte affiché en haut de la fenêtre graphique.
Ajouté à la version 1.0.0.

deltaTime

Variable cacher pour permettre d'actualiser deltaTime.
Ajouté à la version 1.0.0.

fpsCount

Nombre de frames depuis la dernière update de fps.
Ajouté à la version 1.0.0.

fpsDuration

Nombre de secondes depuis la dernière update de fps.
Ajouté à la version 1.0.0.

_lastOverflightedWidget

Widget survolé pendant la dernière frame.
Ajouté à la version 2.0.0.

_modifiedWidget

Liste de tous les widgets modifiés à la dernière frame dans frameEvent pour leur appliquer resetWidget().
Ajouté à la version 1.0.0.

pygameWindow

Fenêtre pygame de base ou effectuer les modifications.
Ajouté à la version 1.0.0.

_widgets

Liste de tous les widgets présents dans le MApp.
Ajouté à la version 1.0.0.
