MScrollArea

MScrollArea est une classe secondaire pour définir une zone qui peut contenir un widget "widgetToScroll" et scroller ce widget.
Cette classe utilise MSlider pour permettre le scroll dans le widget.
Elle hérite de MFrame.
Ajouté à la version 2.0.0.

Fonctions

__init__()

Paramètres : x (assignée à la variabe x de MWidget), y (assignée à la variabe y de MWidget), width (assignée à la variable width de MWidget), height (assignée à la variable height de MWidget), parent (assignée à la variable parent de MWidget), widgetType (assignée à la variable _type de MWidget, basiquememnt mit à "MButton")
Construit un objet MButton.
Ajouté à la version 2.0.0.

getSliderOrientationLength()

Paramètres : aucun
Retourne la largeur d'orientation des sliders.
Ajouté à la version 2.0.0.

getHorizontalSlider()

Paramètres : aucun
Retourne la slider horizontal en référence.
Ajouté à la version 2.0.0.

getVerticalSlider()

Paramètres : aucun
Retourne la slider vertical en référence.
Ajouté à la version 2.0.0.

getWidgetToScroll()

Paramètres : aucun
Retourne le widget scrollant dans le curseur.
Ajouté à la version 2.0.0.

placeSlider()

Paramètres : aucun
Configure les sliders selon le widgetToScroll actuel.
Modifié à la version 2.0.1.

reload()

Paramètres : aucun
Recharge le widget en rappelant setWidgetToScroll() et placeSlider.
Ajouté à la version 2.0.1.

setWidgetToScroll()

Paramètres : widgetToScroll (nouveau widget scrollant dans le curseur)
Change le widget scrollant dans le curseur par "widgetToScroll".
Change aussi le parent de "widgetToScroll" pour le MScrollArea et met setIgnoreUserEvent à True.
Modifié à la version 2.2.0.

_isKeyGettingDropped()

Paramètres : key (touche lachée)
Fonction appelé lorsque le widget est focus est qu'une touche du clavier est lachée.
Permet de mettre _shiftPressed à False.
Ajouté à la version 2.2.0.

_isKeyGettingPressed()

Paramètres : key (touche pressée)
Fonction appelé lorsque le widget est focus est qu'une touche du clavier est pressée.
Permet de mettre _shiftPressed à True.
Ajouté à la version 2.2.0.

_isNotFocusedAnymore()

Paramètres : aucun
Fonction appelé lorsque le widget n'est plus focus.
Permet de mettre _shiftPressed à False.
Ajouté à la version 2.2.0.

_lastUpdate()

Paramètres : deltaTime (temps passé entre cette frame et la dernière frame)
Fonction appelée à chaque frame après le contrôle des évènements et des actions utilisateurs, juste avant frameGraphics.
Dans MScrollArea, elle sert à recharger la zone si widgetToScroll change de taille.
Ajouté à la version 2.0.1.

_lateUpdate()

Paramètres : deltaTime (temps passé entre cette frame et la dernière frame)
Fonction appelée à chaque frame après le contrôle des éléments.
Dans MScrollArea, elle sert à ajuster la position de widgetToScroll selon la valeur des curseurs.
Ajouté à la version 2.0.0.

_mouseWheel()

Paramètres : rotation (valeur de la rotation de la molette)
Fonction appelé lorsque le widget est focus est que l'utilisateur tourne la molette.
Dans MSrollArea, elle transmet le mouvement au slider vertical.
Ajouté à la version 2.2.0.

Variables

sliderOrientationLength

Taille de horizontalSlider et verticalSlider sur leur largeur d'orientation, basiquement à 15.
Ajouté à la version 2.0.0.

horizontalSlider

slider horizontal dans le MScrollArea.
Si aucune slider n'est nécessaire, horizontalSlider n"est alors pas visible.
Ajouté à la version 2.0.0.

verticalSlider

slider vertical dans le MScrollArea.
Si aucune slider n'est nécessaire, verticalSlider n"est alors pas visible.
Ajouté à la version 2.0.0.

widgetToScroll

Widget contenu dans le MScrollArea qui permet le scroll.
Ajouté à la version 2.0.0.

_shiftPressed

Booléen qui indique si la touche "shift" est pressée.
Ajouté à la version 2.2.0.

widgetToScrollOffset

Décalage obligatoire sur widgetToScroll.
Ajouté à la version 2.0.0.
