MScrollArea

MScrollArea est une classe secondaire pour définir une zone qui peut contenir un widget "widgetToScroll" et scroller ce widget.
Cette classe utilise MBar pour permettre le scroll dans le widget.
Elle hérite de MFrame.
Ajouté à la version 2.0.0.

Fonctions

__init__()

Paramètres : x (assignée à la variabe x de MWidget), y (assignée à la variabe y de MWidget), width (assignée à la variable width de MWidget), height (assignée à la variable height de MWidget), parent (assignée à la variable parent de MWidget), widgetType (assignée à la variable _type de MWidget, basiquememnt mit à "MButton")
Construit un objet MButton.
Ajouté à la version 2.0.0.

getBarOrientationLength()

Paramètres : aucun
Retourne la largeur d'orientation des barres.
Ajouté à la version 2.0.0.

getHorizontalBar()

Paramètres : aucun
Retourne la barre horizontal en référence.
Ajouté à la version 2.0.0.

getVerticalBar()

Paramètres : aucun
Retourne la barre vertical en référence.
Ajouté à la version 2.0.0.

getWidgetToScroll()

Paramètres : aucun
Retourne le widget scrollant dans le curseur.
Ajouté à la version 2.0.0.

placeBar()

Paramètres : aucun
Configure les barres selon le widgetToScroll actuel.
Modifié à la version 2.0.1.

reload()

Paramètres : aucun
Recharge le widget en rappelant setWidgetToScroll() et placeBar.
Ajouté à la version 2.0.1.

setWidgetToScroll()

Paramètres : widgetToScroll (nouveau widget scrollant dans le curseur)
Change le widget scrollant dans le curseur par "widgetToScroll".
Change aussi le parent de "widgetToScroll" pour le MScrollArea.
Modifié à la version 2.0.1.

_lateUpdate()

Paramètres : deltaTime (temps passé entre cette frame et la dernière frame)
Fonction appelée à chaque frame après le contrôle des éléments.
Dans MBar, elle sert à ajuster la position de widgetToScroll selon la valeur des curseurs.
Ajouté à la version 2.0.0.

Variables

barOrientationLength

Taille de horizontalBar et verticalBar sur leur largeur d'orientation, basiquement à 15.
Ajouté à la version 2.0.0.

horizontalBar

Barre horizontal dans le MScrollArea.
Si aucune bar n'est nécessaire, horizontalBar est alors pas visible.
Ajouté à la version 2.0.0.

verticalBar

Barre vertical dans le MScrollArea.
Si aucune bar n'est nécessaire, verticalBar est alors pas visible.
Ajouté à la version 2.0.0.

widgetToScroll

Widget contenu dans le MScrollArea qui permet le scroll.
Ajouté à la version 2.0.0.

widgetToScrollOffset

Décalage obligatoire sur widgetToScroll.
Ajouté à la version 2.0.0.
