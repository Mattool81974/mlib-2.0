MTextInputLine

MTextInputLine est une classe secondaire pour définir une entrée texte en une ligne plus facilement qu'avec MText.
Elle hérite de MText.
Ajouté à la version 2.2.0.

Différences à MText

Cette classe est très similaire à MText à quelques exceptions.

Fonctions

__init__()

Paramètres : x (assignée à la variabe x de MWidget), y (assignée à la variabe y de MWidget), width (assignée à la variable width de MWidget), height (assignée à la variable height de MWidget), parent (assignée à la variable parent de MWidget), widgetType (assignée à la variable _type de MWidget, basiquememnt mit à "MButton")
Construit un objet MTextInputLine.
Ajouté à la version 2.2.0.

getInformationColor()

Paramètres : aucun
Retourne la couleur du texte d'information.
Ajouté à la version 2.2.0.

getInformationText()

Paramètres : aucun
Retourne le texte d'informationafficher avant l'entrée utilisateur pour donner des informations de l'utilité de cette entrée à ce dernier.
Ajouté à la version 2.2.0.

getInputColor()

Paramètres : aucun
Retourne la couleur du texte entré par l'utilisateur.
Ajouté à la version 2.2.0.

setInformationColor()

Paramètres : informationColor (nouvelle couleur du texte d'information)
Change la couleur du texte d'information par "informationColor".
Ajouté à la version 2.2.0.

setInformationText()

Paramètres : informationText (nouveau texte d'informationafficher avant l'entrée utilisateur pour donner des informations de l'utilité de cette entrée à ce dernier)
Change le texte d'informationafficher avant l'entrée utilisateur pour donner des informations de l'utilité de cette entrée à ce dernier par informationText.
Ajouté à la version 2.2.0.

setInputColor()

Paramètres : inputColor (nouvelle couleur du texte entré par l'utilisateur)
Change la couleur du texte entré par l'utilisateur par "inputColor".
Ajouté à la version 2.2.0.

_isGettingMouseDown()

Paramètres : button (nombre entier du bouton de souris clické), relativePos (position relative au widget du curseur de la souris)
Fonction appelé lorsque le widget est clické.
Si button est "1", la touche clické est la touche gauche, si button est "2", la touche clické est la molette, si button est "3", la touche clické est la touche droite.
Dans MTextInputLine, elle sert à enlever informationText du texte quand l'utilisateur veut le modifier.
Ajouté à la version 2.2.0.

_isNotFocusedAnymore()

Paramètres : aucun
Fonction appelé lorsque le widget n'est plus focus.
Dans MTextInputLine, elle sert à remettre informationText du texte quand l'utilisateur déselectionne le MTextInputLine sans laisser de texte.
Ajouté à la version 2.2.0.
