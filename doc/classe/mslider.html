MSlider

MSlider est une classe secondaire pour définir un slider.
Elle hérite de MFrame.
Avant la version 2.1.0, elle s'appelait "MBar".
Ajouté à la version 2.0.0.

Affichage

MSlider hérite de MFrame. Elle contient une petite bordure de 1 de large. Elle peut être horizontale ou verticale selon ORIENTATION. Elle contient un bouton amovible pour changer la valeur dans le MSlider.

Fonctionnement

MSlider permet d'utiliser un bouton amovible pour sélectionner une valeur, comprise entre minValue maxValue. Chaque valeur dans le MSlider est séparé de step unités (pas encore disponible).

Fonctions

Construit un objet MSlider.
Ajouté à la version 2.0.0.

Ajouté à la version 2.0.0.

getOrientationAxisLength()

Retourne height si l'orientation est ORIENTATION_BOTTOM_TO_TOP ou ORIENTATION_TOP_TO_BOTTOM, sinon retourne width.
Ajouté à la version 2.3.0.

getStep()

Paramètres : aucun
Retourne le pas entre chaque valeur dans le MSlider.

setButtonOrientationLength()

Paramètres : buttonOrientationLength (nouvelle longeur du bouton amovible sur l'axe de l'orientation)

setChangeButtonBackgroundColorOnOverflight()

Retourne le booléen qui indique si la couleur d'arrière plan du bouton amovible change au survol du curseur par "changeButtonBackgroundColorOnOverflight".

setMaxValue()

Paramètres : maxValue (nouvelle valeur maximale possible dans le MSlider)
Change la valeur maximale possible dans le MSlider par "maxValue".
Cette valeur doit être supérieur ou égal à minValue et value.
Ajouté à la version 2.0.0.

setMinValue()

Paramètres : minValue (nouvelle valeur minimale possible dans le MSlider)
Change la valeur minimale possible dans le MSlider par "minValue".
Cette valeur doit être sinférieur ou égal à maxValue et value.
Ajouté à la version 2.0.0.

setStep()

Paramètres : step (nouveau pas entre chaque valeur dans le MSlider)
Change le pas entre chaque valeur dans le MSlider par "step" si celui ci est diviseur de maxValue - minValue.
Il change aussi wheelMultiplciator à "step".
Modifié à la version 2.2.0.

setValue()

Paramètres : value (nouvelle valeur actuelle dans le MSlider)
Change la valeur actuelle dans le MSlider par "value".
Modifié à la version 2.0.1.

setWheelMultiplicator()

Paramètres : wheelMultiplicator (nouveau coefficient auquelle multiplier la valeur de la roulette de la souris dans _mouseWheel)
Change la valeur du coefficient auquelle multiplier la valeur de la roulette de la souris dans _mouseWheel par "wheelMultiplicator".
Ajouté à la version 2.2.0.

softResetWidget()

Paramètres : aucun
Effectue un reset du widget sans modifications graphiques.
Dans MSlider, elle met _valueChanged à False.
Ajouté à la version 2.0.0.

_doNotOverflightedEffect()

Paramètres : aucun
Effectue sur le MSlider les actions nécessaires lorsque le MSlider est normal (non survolé).
Ajouté à la version 2.0.0.

_doOverflightedEffect()

Paramètres : aucun
Effectue sur le MSlider les actions nécessaires lorsque le MSlider est survolé.
Ajouté à la version 2.0.0.

_getValueAtPos()

Paramètres : relativePos (position à tester dans le MSlider)
Retourne la valeur du MSlider à la position "relativePos".
Ajouté à la version 2.3.0.

_isGettingMouseDown()

Paramètres : button (nombre entier du bouton de souris clické), relativePos (position relative au widget du curseur de la souris)
Fonction appelé lorsque le widget est clické.
Si button est "1", la touche clické est la touche gauche, si button est "2", la touche clické est la molette, si button est "3", la touche clické est la touche droite.
Met à jour _buttonClicked si nécessaire.
Cette fonction est héritée de MWidget.
Modifié à la version 2.3.0.

_isGettingMouseUp()

Paramètres : button (nombre entier du bouton de souris clické), relativePos (position relative au widget du curseur de la souris)
Fonction appelé lorsque le widget n'est plus clické.
Si button est "1", la touche clické est la touche gauche, si button est "2", la touche clické est la molette, si button est "3", la touche clické est la touche droite.
Met à jour _buttonClicked à "False".
Cette fonction est héritée de MWidget.
Ajouté à la version 2.0.0.

_isGettingOverflighted()

Paramètres : aucun
Fonction appelé lorsque le widget est survolé.
Permet d'appeler _doOverflightedEffect lors du survol du curseur.
Cette fonction est héritée de MWidget.
Ajouté à la version 2.0.0.
=

_isNotOverflightedAnymore()

Paramètres : aucun
Fonction appelé lorsque le widget n'est plus survolé par le curseur.
Permet de savoir quand le widget n'est plus survolé pour appeler _doNotOverflightedEffect.
Cette fonction est héritée de MWidget.
Ajouté à la version 2.0.0.

_isPosOverButton()

Paramètres : relativePos (position à tester)
Retourne si la position "relativePos" est au dessus du bouton amovible.
Ajouté à la version 2.3.0.

_mouseMove()

Paramètres : buttons (touches de la souris clické), pos (nouvelle position de la souris), relativeMove (mouvement de la souris)
Fonction appelé lorsque le widget est focus est que l'utilisateur bouge la souris.
Dans MSlider, elle permet de changer la valeur du slider selon le mouvement de la souris.
Modifié à la version 2.3.0.

_mouseWheel()

Paramètres : rotation (valeur de la rotation de la molette)
Fonction appelé lorsque le widget est focus est que l'utilisateur tourne la molette.
Dans MSlider, elle change la valeur présente dans le Slider en multiplitant rotation par wheelMultiplicator.
Ajouté à la version 2.2.0.

_renderBeforeHierarchy()

Paramètres : surface (surface parent)
Met la surface "surface" à jour avec le widget dessus et retourne "surface" mis à jour.
Dans MSlider, elle s'occupe d'afficher le bouton amovible.
Cette fonction est héritée de MWidget.
Modifié à la version 2.3.0.

Variables statiques

ORIENTATION_BOTTOM_TO_TOP

Valeur de ORIENTATION si le slider va de bas en haut, donc 3.
Ajouté à la version 2.3.0.

ORIENTATION_LEFT_TO_RIGHT

Valeur de ORIENTATION si le slider va de la gauche à droite, donc 0.
Ajouté à la version 2.3.0.

ORIENTATION_RIGHT_TO_LEFT

Valeur de ORIENTATION si le slider va de la droite à gauche, donc 2.
Ajouté à la version 2.3.0.

ORIENTATION_TOP_TO_BOTTOM

Valeur de ORIENTATION si le slider va de haut en bas, donc 1.
Ajouté à la version 2.3.0.

buttonBackgroundColor

buttonBackgroundColorOnOverflight

_buttonOverflighted

Booléen qui indique si le bouton amovible est survolé par le curseur ou non.
Ajouté à la version 2.0.0.

_valueChanged

Booléen qui indique si la valeur dans le MSlider a été modifié pendant cette frame.
Ajouté à la version 2.0.0.
