Versions

Cette page contient toutes les versions de la bibliothèque.
Le système de version fonctionne sous le signe : A.B.C, où A est la version principale (changement majeur), B une sous-version (changement mineur) et C les ajustements de sous versions.
Les versions sont aussi chronologiques.
Il est indiqué sur chaque fonction/classe/variable la version à laquelle elle a étée ajouté/modifié.

2.3.1, parue le 05/11/2023-19:00

-Ajout de isChoiceGettingChanged dans MCheckBox pour savoir quand le choix est changée.

2.3.0, parue le 03/11/2023-15:00

-Ajout de la classe primaire MObject pour créer une base à tous les objets dans MLib.
-Ajout de la classe secondaire MBar pour créer une barre de chargement non modifiable.
-Ajout de la classe secondaire MCheckBox pour ajouter des boutons sélectionnable.
-Ajout de la classe secondaire MChrono pour ajouter un chronomètre.
-Ajout d'un système d'orientation différent dans MSlider (comme celui de MBar).

2.2.0, parue le 06/10/2023-11:30

-Ajout de la classe secondaire MTextInputLine pour créer une entrée texte sur une ligne plus simple.
-Ajout de la variable global "VERSION" pour connaître la version actuelle de MLib.
-Ajout de maxTextLength à MText pour définir une taille de texte maximale.
-Amélioration du scrolling dans MScrollArea par l'ajout de _mouseWheel.
-Amélioration du scrolling dans MSlider par l'ajout de posAtClick et _mouseWheel.
-MBar (ajouté à la version 2.0.0) se renomme MSlider.
-Rendu de step optionnel dans MSlider.

2.1.0, parue le 05/10/2023-11:00

-Ajout de caractère interdit/autorisé dans MText.
-Ajout de ignoreUserEvent pour ignorer les évènements utilisateur dans MWidget.
-Ajout de maxFps dans MApp.
-Ajout de setWidgetToFocus pour pouvoir focus un widget en particulier dans MApp.
-Ajout d'une méthode pour avoir la position de la souris dans MApp.
-Amélioration du fonctionnement de MButton en rajoute "oneFrame" à isGettingLeftClicked() et isGettingRightClicked().
-Modification de la façon dont est généré la bordure dans MFrame avec getFrameRendered().
-Modification du générateur dans MText qui prend en charge des fichiers .ttf (SysFont -> Font).
-Optimisation de _render par l'ajout de .convertAlpha aux surface après rendu dans MWidget.
-Supression des imports implicites dans MLib (que des "import lib" et plus "from lib import *").

2.0.1, parue le 28/09/2023-18:30

-Ajout de delete dans MWidget pour mieux anticiper la suppresion d'un widget.
-Ajout de _lastUpdate dans MWidget, appelée juste avant frameGraphics.
-Ajout de l'anti analiasing dans MText.
-Changement de la classe parent de MScrollArea de MFrame à MWidget.
-Correction de la position des barres dans MScrollArea.

2.0.0, parue le 26/09/2023-20:00

-Ajout de la classe secondaire "MSlider".
-Ajout de la classe secondaire "MButton".
-Ajout de la classe secondaire "MScrollArea".
-Ajout de "_isNotOverflightedAnymore" dans MWidget, appelé quand le widget n'est plus survolé par le curseur.
-Ajout de "_lateUpdate" dans MWdiget, appelé chaque frame après le contrôle des évènments.
-Ajout des classes de catégorie "secondaire".
-Ajotu d'une page de navigation dans la documentation.
-Modification de l'usage de mouseDown et mouseUp dans MWidget.

Cette version est sortie un peu en précipitation, les fonctionnalités listées seront entièrement débuggées dans les versions suivantes.

1.4.0, parue le 22/09/2023-18:30

-Ajout de "_mouseWheel" dans MWidget pour permettre de prendre en charge la roulette de la souris.
-Ajout d'un système de console dans MApp pour faciliter le débugage de MLib.
-Ajout d'un système de navigation (proche d'un système avec des sliders) du texte dans MText.
-Ajout d'une icône de l'application dans MApp.
-Ajout de raccourcis claviers pour la sélection dans les entrées textes normales dans MText.
-Optimisation de l'affichage de MText. -Résolution d'un bug lors de la sélection de texte aligné dans MText.

1.3.0, parue le 15/09/2023-16:00

-Ajout du copie/collage du texte dans MText.
-Ajout de la sélection de texte dans MText.
-Modification de setInput dans MText pour modifier le curseur et la sélection en une seule fonction.
-Optimisation de MText par l'affichage de seulement le texte visible.

1.2.0, parue le 07/09/2023-17:00

-Ajout d'un offset au texte dans MText.
-Résolution d'un bug avec le curseur pour les MText sans coupure dynamique.
-Résolution de bugs avec la bordure et l'alignement du texte dans MText.

1.1.0, parue le 04/09/2023-15:30

-Ajout de l'alignement/position sur les images de MImage.
-Ajout du recadrement/taille sur les images de MImage.

1.0.0, parue le 03/09/2023-17:00

Première version du système de versionning MLib.
Tous les ajouts ne sont pas listés, car ils sont trop nombreux.
