Versions

Cette page contient toutes les versions de la bibliothèque.
Le système de version fonctionne sous le signe : A.B.C, où A est la version principale (changement majeur), B une sous-version (changement mineur) et C les ajustements de sous versions.
Les versions sont aussi chronologiques.
Il est indiqué sur chaque fonction/classe/variable la version à laquelle elle a étée ajouté/modifié.

2.0.0, prévue pour le 09/2023

-Ajout de la classe secondaire "MBar".
-Ajout de la classe secondaire "MButton".
-Ajout de la classe secondaire "MScrollArea".
-Ajout de "_isNotOverflightedAnymore" dans MWidget, appelé quand le widget n'est plus survolé par le curseur.
-Ajout de "_lateUpdate" dans MWdiget, appelé chaque frame après le contrôle des évènments.
-Ajout des classes de catégorie "secondaire".
-Ajotu d'une page de navigation dans la documentation.
-Modification de l'usage de mouseDown et mouseUp dans MWidget.

Cette version est sortie un peu en précipitation, les fonctionnalités listées seront entièrement débuggées dans les versions suivantes.

1.4.0, parue le 22/09/2023-18:30

-Ajout de "_mouseWheel" dans MWidget pour permettre de prendre en charge la roulette de la souris.
-Ajout d'un système de console dans MApp pour faciliter le débugage de MLib.
-Ajout d'un système de navigation (proche d'un système avec des sliders) du texte dans MText.
-Ajout d'une icône de l'application dans MApp.
-Ajout de raccourcis claviers pour la sélection dans les entrées textes normales dans MText.
-Optimisation de l'affichage de MText. -Résolution d'un bug lors de la sélection de texte aligné dans MText.

1.3.0, parue le 15/09/2023-16:00

-Ajout du copie/collage du texte dans MText.
-Ajout de la sélection de texte dans MText.
-Modification de setInput dans MText pour modifier le curseur et la sélection en une seule fonction.
-Optimisation de MText par l'affichage de seulement le texte visible.

1.2.0, parue le 07/09/2023-17:00

-Ajout d'un offset au texte dans MText.
-Résolution d'un bug avec le curseur pour les MText sans coupure dynamique.
-Résolution de bugs avec la bordure et l'alignement du texte dans MText.

1.1.0, parue le 04/09/2023-15:30

-Ajout de l'alignement/position sur les images de MImage.
-Ajout du recadrement/taille sur les images de MImage.

1.0.0, parue le 03/09/2023-17:00

Première version du système de versionning MLib.
Tous les ajouts ne sont pas listés, car ils sont trop nombreux.
